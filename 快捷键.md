### typora快捷键

```
一级标题：⌘1 (command + 1)
二级标题：⌘2 (command + 2)
三级标题：⌘3 (command + 3)

源码模式：⌘/ (command + /)
表格：⌥⌘T (option + command + T)
代码块：⌥⌘C (option + command + C)
引用：⌥⌘Q (option + command + Q)
有序列表：⌥⌘O (option + command + O)
无序列表：⌥⌘U (option + command + U)

任务列表：⌥⌘X (option + command + X)
水平分割线：⇧⌘- (option + command + -)

加粗：⌘B (command + B)
斜体：⌘I (command + I)

大纲视图：⌃⌘1 (control + command + 1)
显示/隐藏侧边栏：⇧⌘L (shift + command + L)
大纲视图：⌃⌘1 (control + command + 1)
文档列表视图：⌃⌘2 (control + command +2)
文件树视图：⌃⌘3 (control + command + 3)

```



### Mac快捷键

```
* 快速打开终端
  1. command+空格，会弹出搜索框，输入term, 回车，就出来了; Or
  2. control+空格，输入term, 回车
  3. 强制退出 option + command + esc
 
```

### VScode快捷键

```
command + p  快速查找，跳转到文件Quick Open
option + command + ←/→	返回/前进 Go back / forward
```



### Mac Idea 快捷键

```
option + command + ←/→	返回/前进 Go back / forward
commond + F12 (⌘ + F12) 显示类的方法
control + command + U  单词翻译（需要安装translation翻译插件）
commond + O  搜索java文件
control + option + B ： 打开接口方法具体实现 | 查看继承关系
option + enter ： 自动修复错误 
option + command + L ： 格式代码
option + F7 ： 查看方法的所有直接调用者
Option + F3 选中文件/文件夹/代码行，使用助记符添加/取消书签
Command + F3 显示所有书签
Shift + F12 全屏显示


```
